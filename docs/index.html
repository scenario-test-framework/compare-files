01 / TEXT

行ではなく、項目で差分を見る

比較キーで対応する行を特定し、セル単位の差分をCSVに出力。並び順が違うファイルも比較できます。

customer_idamountstatus
1004212,800 → 13,200paid
100438,400paid
⌗

02 / IMAGE

画像差分を赤枠で特定

ピクセル差分を左右比較のPNGへ。動的な領域は除外エリアとして指定できます。

⌁

03 / BATCH

ディレクトリをまとめて比較

サブディレクトリを再帰的に走査。ファイル形式に応じて比較方法を切り替え、サマリーを生成します。

✓ orders.csv✓ profile.json× receipt.png
›_

04 / AUTOMATION

CI/CDに、そのまま組み込める

成功 0・差分あり 3・エラー 6 の明確な終了コード。ローカルとCIで同じ比較を再現できます。

$ compare_files expected actual
Compare status: NG
  File: 24  OK: 22  NG: 2
$ echo $?
